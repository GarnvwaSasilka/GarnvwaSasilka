## Featured Project: PhishDetect AI

A three-class phishing email classifier (legitimate / traditional phishing / AI-generated phishing) — built end-to-end from model to deployment.

- 98.2% accuracy using TF-IDF + Logistic Regression on 15,000 samples
- LIME explainability and Cialdini persuasion-tactic detection for interpretable predictions
- Full-stack delivery: Streamlit web app, SQLite scan history, PDF reporting, Chrome extension for Gmail, Railway-hosted API
- Synthetic AI-phishing samples generated with DistilGPT2 for robustness testing

Live Demo: https://phishdetectai-qkhobpytynh7wu25zv2c9n.streamlit.app
